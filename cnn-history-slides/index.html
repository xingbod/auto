CNN 发展历程

从感受野与 Neocognitron，到 LeNet、AlexNet、VGG、Inception、ResNet、DenseNet、MobileNet、EfficientNet 与 ConvNeXt：经典工作、作者背景、网络架构与演化主线。

操作：方向键 / 空格翻页，P 打印或导出 PDF。

授课目标：不只是“列模型”，而是看清发展逻辑

三条主线

  1. 归纳偏置：局部连接、权值共享、平移等变。
  2. 可训练深度：数据、算力、初始化、归一化、残差连接。
  3. 效率与可部署：参数量、FLOPs、延迟、硬件友好。

CNN 的历史不是单线“更深更大”，而是围绕表示能力、优化难度、计算效率之间的持续折中。

CNN 发展时间线：从神经科学到工程系统

1959–1968Hubel & Wiesel 发现视觉皮层简单/复杂细胞；感受野与层级视觉表示成为 CNN 的神经科学灵感。
1980Fukushima 提出 Neocognitron：S/C 层交替，奠定卷积与池化思想雏形。
1989–1998LeCun 等用反向传播训练卷积网络，LeNet-5 成为早期工业化成功案例。
2012AlexNet 在 ImageNet 突破，深度 CNN 进入现代计算机视觉主舞台。
2013–2015ZFNet 可视化解释、VGG 统一 3×3 深层堆叠、GoogLeNet 用 Inception 多尺度稀疏近似。
2015–2017BatchNorm、ResNet、DenseNet、SENet 等解决深层训练与通道注意力。
2017–2022MobileNet/EfficientNet/ConvNeXt：从移动端效率到 CNN 与 Transformer 时代的再设计。

背景奠基：感受野与层级视觉系统

工作

Receptive fields, binocular interaction and functional architecture in the cat's visual cortex（Hubel & Wiesel, 1962）等系列研究。

作者简介：David H. Hubel 与 Torsten N. Wiesel 是神经生理学家，因研究视觉皮层信息处理获得 1981 年诺贝尔生理学或医学奖。

对 CNN 的影响

  • 局部感受野：神经元只响应局部空间区域。
  • 简单细胞/复杂细胞：边缘检测与位置容忍。
  • 层级组合：低级边缘 → 局部形状 → 高级物体。
局部边缘→局部不变性→层级组合→物体识别

脉络意义：CNN 把视觉皮层的“局部、共享、层级”转化为可微分、可训练的工程架构。

1980：Neocognitron

标题：Neocognitron: A self-organizing neural network model for a mechanism of pattern recognition unaffected by shift in position

作者：Kunihiko Fukushima

作者简介：日本计算神经科学与神经网络研究先驱，长期研究受视觉皮层启发的模式识别模型。

核心贡献：提出 S-cell 与 C-cell 交替结构，分别近似特征检测与位置容忍，是现代卷积层/池化层的早期原型。

网络架构

输入→S1 特征检测→C1 位移容忍→S2→C2→分类
  • S 层：学习局部特征模板。
  • C 层：对邻域 S 响应进行汇聚，增强平移不变性。
  • 训练主要是自组织/局部学习，不是现代端到端反向传播。

脉络意义：结构思想领先，但缺少大数据、算力与反向传播训练体系。

1989/1998：LeNet 系列与 LeNet-5

标题：Backpropagation Applied to Handwritten Zip Code Recognition（1989）；Gradient-Based Learning Applied to Document Recognition（1998）

作者：Yann LeCun、Léon Bottou、Yoshua Bengio、Patrick Haffner 等

作者简介：LeCun 是深度学习与 CNN 代表人物，图灵奖得主；Bengio 同为深度学习图灵奖得主；Bottou 长期研究大规模优化与机器学习系统。

核心贡献：用反向传播端到端训练卷积网络，在手写数字、支票识别等任务上形成可部署系统。

LeNet-5 架构

32×32 输入→C1: 6@5×5→S2: avg pool→C3: 16@5×5→S4→C5: 120→F6: 84→输出
  • 局部连接 + 权值共享降低参数量。
  • 池化带来小范围形变鲁棒性。
  • 全连接层完成类别决策。

脉络意义：CNN 第一次被证明可以端到端训练并解决真实工业问题。

1995–1998：TDNN 与早期卷积思想扩展

标题：Phoneme Recognition Using Time-Delay Neural Networks（Waibel et al., 1989）等

作者：Alexander Waibel、Toshiyuki Hanazawa、Geoffrey Hinton 等

作者简介：Waibel 是语音识别与多模态交互专家；Hinton 是深度学习图灵奖得主，长期推动神经网络表示学习。

架构思想

时间序列输入→共享时间窗滤波→更高层时序组合→音素/词分类
  • 在时间维上使用共享权重，类似一维卷积。
  • 说明“局部共享”不仅适用于图像，也适用于序列。

脉络意义：卷积是一种可迁移的结构先验，可用于空间、时间和多维信号。

2012：AlexNet 引爆现代深度视觉

标题：ImageNet Classification with Deep Convolutional Neural Networks

作者：Alex Krizhevsky、Ilya Sutskever、Geoffrey E. Hinton

作者简介：Krizhevsky 是深度学习工程先驱；Sutskever 后来成为 OpenAI 联合创始人与首席科学家；Hinton 是深度学习奠基人之一、图灵奖得主。

核心贡献：在 ImageNet 2012 大幅降低错误率，证明“大数据 + GPU + 深网络 + 非饱和激活”的组合可行。

AlexNet 架构

224×224→11×11/s4, 96→LRN+Pool→5×5, 256→Pool→3×3, 384→3×3, 384→3×3, 256→4096→4096→1000
  • ReLU 加速训练；Dropout 抑制过拟合。
  • 数据增强与 GPU 并行训练关键。
  • 早期使用 LRN，后续被 BatchNorm 等取代。

脉络意义：CNN 从“特定任务模型”变成通用视觉骨干网络。

2013：ZFNet —— 可视化推动结构改进

标题：Visualizing and Understanding Convolutional Networks

作者：Matthew D. Zeiler、Rob Fergus

作者简介：Zeiler 是视觉深度学习研究者，后创办 Clarifai；Fergus 是 NYU/Meta AI 研究者，研究视觉表示学习。

核心贡献：通过反卷积可视化理解各层特征，发现 AlexNet 第一层步长和卷积核过大导致信息损失，并据此改进网络。

ZFNet 架构相对 AlexNet

224×224→7×7/s2→Pool→5×5/s2→Pool→3×3 堆叠→FC 分类
  • 缩小首层卷积核：11×11 → 7×7。
  • 减小首层步长：4 → 2。
  • 强调可解释性可直接反馈架构设计。

脉络意义：CNN 研究开始从“堆模型”走向“理解模型行为后再设计”。

2013/2014：Network in Network 与 1×1 卷积

标题：Network in Network

作者：Min Lin、Qiang Chen、Shuicheng Yan

作者简介：Shuicheng Yan 长期研究计算机视觉与机器学习；该工作来自新加坡国立大学团队。

核心贡献：用 micro neural network 替代线性卷积滤波器，引入 1×1 卷积进行通道混合，并用全局平均池化减少全连接层。

NIN 模块架构

k×k Conv→1×1 Conv→1×1 Conv→Global Avg Pool→Softmax
  • 1×1 卷积成为 Inception、ResNet bottleneck、MobileNet 等后续架构的基础组件。
  • 全局平均池化让空间特征直接对应类别响应，减少参数与过拟合。

脉络意义：网络设计从“空间卷积”扩展到“通道维度的信息重组”。

2014：VGG —— 简洁、深、可迁移

标题：Very Deep Convolutional Networks for Large-Scale Image Recognition

作者：Karen Simonyan、Andrew Zisserman

作者简介：二者来自牛津大学 Visual Geometry Group；Zisserman 是计算机视觉领域代表学者，长期研究几何视觉、识别与视频理解。

核心贡献：证明使用统一的小卷积核 3×3、简单规则地加深网络，可以获得强表示和良好迁移能力。

VGG-16 架构

224×224→2×[3×3,64]→Pool→2×[3×3,128]→Pool→3×[3×3,256]→Pool→3×[3×3,512]→Pool→3×[3×3,512]→4096-4096-1000
  • 两个 3×3 近似一个 5×5 感受野，三个 3×3 近似一个 7×7，但参数更少且非线性更多。
  • 缺点：全连接层参数巨大，计算和存储成本高。

脉络意义：VGG 成为“CNN backbone”的教学与迁移学习模板。

2014：GoogLeNet / Inception —— 多尺度与降维

标题：Going Deeper with Convolutions

作者：Christian Szegedy、Wei Liu、Yangqing Jia、Pierre Sermanet、Scott Reed、Dragomir Anguelov、Dumitru Erhan、Vincent Vanhoucke、Andrew Rabinovich 等

作者简介：Szegedy 是 Google Research 研究者，推动 Inception 系列；Yangqing Jia 是 Caffe 作者；团队来自 Google 与学术/工业视觉社区。

核心贡献：Inception 模块在同一层并行使用多种卷积尺度，并用 1×1 卷积做瓶颈降维。

Inception v1 模块

输入→1×11×1→3×31×1→5×5Pool→1×1→Concat
  • 多尺度：同时捕捉局部与较大感受野。
  • 瓶颈：1×1 降低通道数，控制计算量。
  • 辅助分类器：缓解深层梯度问题。

脉络意义：从单一路径堆叠转向“模块化、多分支、计算预算受控”的网络设计。

2015：Batch Normalization —— 深网训练稳定器

标题：Batch Normalization: Accelerating Deep Network Training by Reducing Internal Covariate Shift

作者：Sergey Ioffe、Christian Szegedy

作者简介：Ioffe 是 Google 研究者，研究机器学习与优化；Szegedy 同时是 Inception 系列核心作者。

核心贡献：在小批量维度归一化中间激活，允许更大学习率、加速收敛，并提供正则化效果。

典型 Conv-BN-ReLU 模块

Conv→BatchNorm→ReLU
  • 成为 ResNet、Inception-v2/v3 等网络的标配。
  • 把“训练技巧”提升为“架构基本层”。
  • 推理时使用滑动平均统计量。

脉络意义：CNN 的发展不仅靠新拓扑，也靠优化与归一化机制。

2015：ResNet —— 残差学习解决退化问题

标题：Deep Residual Learning for Image Recognition

作者：Kaiming He、Xiangyu Zhang、Shaoqing Ren、Jian Sun

作者简介：何恺明是计算机视觉代表学者，后在 FAIR/MIT 等机构研究视觉基础模型；孙剑是微软亚洲研究院杰出研究者；团队长期推动检测、分割和深度视觉基础组件。

核心贡献：学习残差 F(x)=H(x)-x，通过 identity shortcut 让极深网络易优化。

残差块与 ResNet-50

x→1×1→3×3→1×1+shortcut x→ReLU
7×7 stem→Pool→conv2_x ×3→conv3_x ×4→conv4_x ×6→conv5_x ×3→GAP→FC
  • 34/50/101/152 层成为标准深度。
  • 后续检测、分割、姿态估计长期使用 ResNet backbone。

脉络意义：残差连接成为深度学习通用架构范式，不限于 CNN。

2016：DenseNet —— 特征复用与密集连接

标题：Densely Connected Convolutional Networks

作者：Gao Huang、Zhuang Liu、Laurens van der Maaten、Kilian Q. Weinberger

作者简介：黄高来自清华/康奈尔研究背景，研究高效视觉网络；van der Maaten 是 t-SNE 作者之一；Weinberger 长期研究机器学习与表示学习。

核心贡献：每一层接收此前所有层的特征作为输入，显式增强特征复用和梯度流。

Dense Block 架构

x0→H1concatH2concatH3concatTransition: 1×1 + Pool
  • 增长率 growth rate 控制每层新增通道。
  • 相比单纯加深，参数效率更高。
  • 代价是连接和显存访问更复杂。

脉络意义：深层网络的核心问题变为“信息和梯度如何跨层流动”。

2016–2017：ResNeXt 与“基数”维度

标题：Aggregated Residual Transformations for Deep Neural Networks

作者：Saining Xie、Ross Girshick、Piotr Dollár、Zhuowen Tu、Kaiming He

作者简介：Girshick 是 R-CNN 系列核心作者；Dollár 是视觉检测与特征学习专家；何恺明是 ResNet 作者之一。

核心贡献：提出 cardinality（分支/组数）是除深度、宽度之外的重要容量维度，用 grouped convolution 聚合多个残差变换。

ResNeXt block

x→1×1→3×3 grouped conv, C=32→1×1+shortcut
  • 统一理解 Inception 多分支与 ResNet 残差。
  • 组卷积让模型容量增加但计算可控。

脉络意义：CNN 设计开始系统探索深度、宽度、分支数三者的扩展规律。

2017：SENet —— 通道注意力

标题：Squeeze-and-Excitation Networks

作者：Jie Hu、Li Shen、Gang Sun

作者简介：该团队来自 Momenta 等机构；SENet 获得 ILSVRC 2017 分类任务冠军，推动注意力机制进入 CNN 主干设计。

核心贡献：通过全局信息自适应重标定通道权重，使网络学会“强调哪些通道”。

SE 模块

特征 U→Squeeze: GAP→FC r↓ + ReLU→FC + Sigmoid→Scale 通道
  • 可插入 ResNet、Inception、ResNeXt 等模型。
  • 参数/计算开销很小，但精度提升明显。

脉络意义：CNN 从固定卷积算子走向输入自适应的注意力调制。

2016–2017：SqueezeNet 与 MobileNet —— 轻量化 CNN

SqueezeNet

标题：SqueezeNet: AlexNet-level accuracy with 50x fewer parameters and <0.5MB model size

作者：Forrest N. Iandola、Song Han、Matthew W. Moskewicz、Khalid Ashraf、William J. Dally、Kurt Keutzer

作者简介：Song Han 与 Dally 长期研究高效深度学习与硬件；Keutzer 研究并行计算和机器学习系统。

Squeeze 1×1→Expand 1×1Expand 3×3→Concat

MobileNet v1

标题：MobileNets: Efficient Convolutional Neural Networks for Mobile Vision Applications

作者：Andrew G. Howard、Menglong Zhu、Bo Chen、Dmitry Kalenichenko、Weijun Wang、Tobias Weyand、Marco Andreetto、Hartwig Adam

作者简介：Google 团队，聚焦移动端视觉模型与产品级部署。

Depthwise 3×3→Pointwise 1×1→BN/ReLU

脉络意义：当 CNN 进入手机与嵌入式设备，评价标准从单一精度扩展为精度-参数-延迟-能耗。

2017：Xception —— 极致 Inception 与深度可分离卷积

标题：Xception: Deep Learning with Depthwise Separable Convolutions

作者：François Chollet

作者简介：Chollet 是 Keras 作者，长期推动深度学习软件生态和模型设计研究。

核心贡献：把 Inception 理解为通道相关性与空间相关性分解；使用 depthwise separable convolution 进一步解耦空间卷积和通道混合。

Xception 模块

输入→Depthwise spatial conv→Pointwise 1×1 conv→Residual shortcut
  • 与 MobileNet 共享高效卷积思想，但面向更强表示。
  • 说明卷积可以被结构化分解以提升效率。

脉络意义：可分离卷积成为高效 CNN 的核心算子之一。

2019：EfficientNet —— 复合缩放规律

标题：EfficientNet: Rethinking Model Scaling for Convolutional Neural Networks

作者：Mingxing Tan、Quoc V. Le

作者简介：Tan 与 Le 来自 Google Brain；Quoc Le 长期研究 AutoML、神经架构搜索和大规模深度学习。

核心贡献：用 NAS 找到 EfficientNet-B0，再用 compound scaling 同时按比例缩放深度、宽度、输入分辨率。

EfficientNet 架构

Stem Conv→MBConv1→MBConv6 + SE→多阶段重复→GAP→FC
  • MBConv：MobileNetV2 inverted residual + depthwise conv。
  • SE 模块提高通道自适应能力。
  • 复合缩放：depth α^φ, width β^φ, resolution γ^φ。

脉络意义：CNN 设计进入“结构搜索 + 缩放定律 + 硬件效率”的阶段。

2022：ConvNeXt —— Transformer 时代重新审视 CNN

标题：A ConvNet for the 2020s

作者：Zhuang Liu、Hanzi Mao、Chao-Yuan Wu、Christoph Feichtenhofer、Trevor Darrell、Saining Xie

作者简介：团队来自 Meta AI/UC Berkeley 等机构；Saining Xie 也是 ResNeXt 作者之一，长期研究视觉架构。

核心贡献：借鉴 Swin Transformer 的训练与宏观设计，用现代化改造证明纯 CNN 仍可达到强竞争力。

ConvNeXt block

7×7 Depthwise Conv→LayerNorm→1×1 / Linear 扩张4×→GELU→1×1 / Linear 降维+shortcut
  • 更大卷积核、倒残差、LayerNorm、GELU、少激活函数。
  • 把 CNN 与 Transformer 的设计语言拉近。

脉络意义：CNN 不是被 Transformer 简单替代，而是在新训练范式下重新进化。

发展脉络总结：CNN 的“设计语法”如何演化

1. 从神经启发到端到端学习

Neocognitron 提供 S/C 层结构直觉；LeNet 把卷积、池化、反向传播结合为可训练系统。

2. 从浅层到深层

AlexNet 证明大规模深度 CNN 可行；VGG 让“更深 + 小卷积核”成为清晰模板。

3. 从单路径到模块化

Inception、ResNeXt 引入多分支、1×1、组卷积，在表达能力和计算预算间折中。

4. 从堆叠到跨层信息流

BatchNorm、ResNet、DenseNet 重点解决梯度传播、退化和特征复用。

5. 从精度到部署效率

SqueezeNet、MobileNet、EfficientNet 推动轻量化、可分离卷积、缩放策略和硬件意识。

6. 从固定算子到自适应与现代化

SENet 的通道注意力、ConvNeXt 的现代训练与结构重构说明 CNN 仍持续吸收新范式。

课堂讲解建议：如何串联这些工作

  1. 先讲问题：图像维度高，普通 MLP 参数爆炸；CNN 用局部连接和权值共享注入图像先验。
  2. 再讲三次跃迁：LeNet 的可训练、AlexNet 的规模化、ResNet 的极深化。
  3. 然后讲设计分叉：VGG 走简单堆叠，Inception 走多尺度模块，ResNet 走跨层连接。
  4. 最后讲现实约束：模型不只是准，还要快、小、省电、易部署。
  5. 引向今天：Transformer 出现后，CNN 的归纳偏置仍在检测、分割、移动端和混合架构中重要。

一句话主线：CNN 的历史，就是把“视觉先验”不断转化为“可优化、可扩展、可部署”的网络结构。
